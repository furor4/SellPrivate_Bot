# SellPrivate Bot

Telegram-бот для автоматизации платных подписок на приватный канал или чат с оплатой через **Telegram Stars**.

## Команды

1. `/start` - Запускает бота и предлагает получить доступ к приватному ресурсу

2. `/info` - Предоставляет информацию о текущей подписке и сроке её действия

3. `/stats` - Показывает статистику продаж _(Доступно только для владельца)_

4. `/setprice <тариф>(month, sixmonth, year) <цена>` - Установка цены тарифа _(Доступно только для владельца)_

## Особенности
* **Оплата через Telegram Stars** — официальная и безопасная монетизация.
* **Автоматические инвайты** — бот сам генерирует уникальную одноразовую ссылку после оплаты.
* **Контроль подписки** — встроенный планировщик задач (`apscheduler`) автоматически проверяет сроки, напоминает за сутки до окончания и кикает просроченных пользователей.
* **Админ-панель** — просмотр статистики и быстрое изменение цен прямо из чата.

## Инструкция по запуску

### 1. Клонирование проекта
```bash
git clone https://github.com/furor4/SellPrivate_Bot.git
cd SellPrivate_Bot
```

### 2. Настройка окружения (`.env`)

Переименуйте файл `.env.example` в корне проекта в `.env` и заполните свои данные.

```ini
TOKEN='токен_вашего_бота'
ADMIN=телеграм_айди_админа
PRIVATE_CHANNEL_ID=айди_канала
PRIVATE_CHAT_ID=айди_чата (если существует, иначе оставьте пустым)

DB_USER='postgres'
DB_PASSWORD='пароль_от_бд'
DB_NAME='sub_bot_db'
```

### 3. Запуск через Docker Compose

Соберите и запустите контейнеры бота и базы данных в фоновом режиме:

```bash
docker compose up -d --build
```
